# Sistema de Controle de Pontos

Sistema profissional de controle de pontos desenvolvido com Next.js e integração com Google Sheets.

## Funcionalidades

- ✅ Autenticação segura com Google OAuth
- ✅ Dashboard profissional
- ✅ Registro de entrada e saída
- ✅ Histórico de pontos
- ✅ Integração com Google Sheets
- ✅ Interface responsiva

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- NextAuth.js
- Google APIs
- Material-UI

## Instalação

1. Extrair arquivo ZIP
2. `npm install`
3. Configurar `.env.local`
4. `npm run dev`
